# Our Work:
The primary objective of this study is to explore the application of Grover’s Algorithm to structured problems inspired by braid groups. Here, we focus on identifying braid words that produce specified strand permutations. By modeling braid transformations as a structured search problem over sequences of generators, we evaluate the efficiency of quantum search in recovering hidden
input sequences. Each word is mapped not to a full braid invariant, but to the induced permutation in S3. Thus, the oracle marks those words whose induced permutation
matches a given target permutation. This constitutes a many-to-one pre-image search under the quotient map B3 → S3, rather than full braid reconstruction.
